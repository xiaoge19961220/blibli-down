# BiliArchiver 隐私政策与安全声明 (Privacy Policy)

本隐私政策适用于 **BiliArchiver (哔哩归档大师)** 应用程序（包括其桌面版和网页版）。

我们深知个人隐私和账号凭证安全的重要性。本声明详细说明了我们如何处理您的数据：

### 1. 核心安全承诺
- **100% 纯本地运行**：BiliArchiver 的所有网络请求、数据解析、多线程下载和媒体转码流程均在您的**个人设备（本地）**上运行。
- **无第三方中转服务器**：本软件不设任何后台云服务器，您的所有操作（如视频检索、账号登录、视频流提取）直接与哔哩哔哩（Bilibili）官方服务器通信。
- **无数据收集与上传**：我们承诺**绝不收集、绝不存储、绝不上传**您的任何个人隐私数据、浏览器历史、Cookie凭证、下载记录或媒体内容。

### 2. 敏感凭证 (SESSDATA / Cookie) 处理
- 为了保障您下载高清视频（如 1080P、4K 等）的权益，软件支持通过 Bilibili 官方扫码或手动配置 `SESSDATA`。
- 您的登录凭证（Cookie）**仅加密保存在您本地**的配置文件（`config.json` 或 `localStorage`）中。
- 凭证仅在向哔哩哔哩官方 API 请求视频播放地址时作为 HTTP Headers 中的 `Cookie` 字段发送。我们绝不会将您的任何 Cookie 传输给任何第三方。

### 3. 文件及存储权限
- 软件需要访问您指定的**本地保存目录**，仅用于存储您主动发起下载的视频、音频和弹幕文件。
- 软件不会擅自扫描、修改或上传您磁盘中的其他无关文件。

### 4. 在线升级服务
- 软件自带的检查更新功能，是通过向公认的 **GitHub API** 检索本项目最新 Release 版本号实现的。
- 该过程不会上传您的任何设备标识符或个人信息，仅用于比较本地版本并拉取 GitHub Actions 官方签名的安装包。
